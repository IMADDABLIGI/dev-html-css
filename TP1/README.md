# Site Web Innosys - TP1

## Structure du Site

Ce dossier contient le site web pour la société Innosys, spécialisée dans le développement d'applications informatiques.

### Fichiers HTML

1. **index.html** - Page d'accueil avec menu de navigation
2. **propos.html** - Présentation de l'entreprise
3. **produits.html** - Catalogue des produits et services
4. **contact.html** - Informations de contact et formulaire

### Dossier Images

Le dossier `images/` contient toutes les images utilisées :
- logo.jpg - Logo de l'entreprise
- propos.jpg - Image de présentation
- map.JPG - Carte de localisation
- vente_assurence.JPG - Image supplémentaire

## Comment utiliser le site

1. **Ouvrir le site** : Double-cliquez sur `index.html` pour ouvrir la page d'accueil dans votre navigateur
2. **Navigation** : Utilisez le menu pour accéder aux différentes pages
3. **Retour** : Chaque page contient un lien "Retour" pour revenir à l'accueil

## Caractéristiques

✅ Design moderne et responsive
✅ Navigation intuitive avec liens fonctionnels
✅ Lien "Retour" sur chaque page secondaire
✅ Images intégrées
✅ Style CSS cohérent
✅ Formulaire de contact
✅ Carte de localisation
✅ Information complète sur l'entreprise

## Technologies utilisées

- HTML5
- CSS3 (styles intégrés)
- Design responsive
